Project Overview

Arnotes is a revolutionary education platform leveraging AI technologies like BERT and DeepSpeech to redefine note-taking. The primary target audience is students, starting with IB students and gradually expanding to all high school students. The platform will automatically transcribe lectures, highlight important parts, and offer tools for classifying notes, generating flashcards, and self-testing.

Technology Stack

Our technology stack consists of Python for backend development, React for frontend, and BERT and DeepSpeech models for AI capabilities. We are using NER models like SpaCy and BERT-NER to enhance the note-taking functionality. The platform also integrates natural language processing libraries for BERT-based NER model training and prediction.

Detailed Breakdown

In the design phase, we conducted research, defined the visual style, and created wireframes and mockups. For backend development, we set up a server environment, created database models, and implemented user authentication and API endpoints. We collected and annotated a dataset of IB history-related text for model training. The frontend was built with React and provides an interactive interface integrated with the backend API.

What's Next?

We're currently in the early testing phase with students at Ivanhoe Grammar. Our next step is to expand to all IB students, and then eventually all high schoolers and students. We also plan to add more features like classifying notes, generating flashcards, and self-testing tools. Stay tuned for more updates!
